Utilizando Links

Os links pode ser gerados utilizando anchors (ancoras) que são definidas pela tag <a>. Essa tag tem alguns parametros como href= que diz qual é o link que deve ser coloca na palavra escolida e tager= que define o modo de abertura.


Links Externos

Links externos são links que levam para outros sites como para o youtube.

Links internos

Links internos são os links que levam para uma pagina dentro do própio site como a pagina home.

Links de download

  • Download
