Audio

Os audios podem ser inseridos nas paginas html por meio da tag <audio> seguindo a seguinte estrutura:

Código;


<audio controls>
    <source src="media/Sleep Away.mp3" type="audio/mpeg">
    <p>Esse paragrafo aparece apenas se seu navegador não der suporte ao tipo de audio especificado, você pode fazer o download <a href="media/Sleep Away.mp3"download type="audio/mpeg" >clicando aqui</a>.</p>
</audio>

Note que é possivel colocar varios sources com diferentes formatos de audio.
